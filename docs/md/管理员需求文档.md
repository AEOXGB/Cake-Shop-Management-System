
# 蛋糕系统 - 管理员功能需求文档

## 1. 需求概述

基于当前蛋糕系统已实现的用户端功能（商品浏览、购物车、订单管理等），管理员需要相应的后台管理功能来维护系统运营。

## 2. 功能需求清单

### 2.1 管理员登录模块

| 需求编号 | 功能名称 | 需求描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| ADMIN-001 | 管理员登录 | 管理员通过账号密码登录后台管理系统 | 高 |
| ADMIN-002 | 安全退出 | 管理员退出登录状态，清空session | 高 |

### 2.2 用户管理模块

| 需求编号 | 功能名称 | 需求描述 | 优先级 |
| :--- | :--- | :--- | :--- |

| ADMIN-004 | 用户详情 | 查看单个用户的详细信息（用户名、电话、地址等） | 中 |
| ADMIN-005 | 用户搜索 | 根据用户名或电话搜索用户 | 中 |
| ADMIN-006 | 用户状态管理 | 禁用/启用用户账号 | 高 |

### 2.3 商品管理模块

| ADMIN-011 | 商品搜索 | 根据商品名称搜索 | 中 |

| :--- | :--- | :--- | :--- |

| ADMIN-014 | 添加分类 | 添加新的商品分类 | 高 |

## 3. 业务流程

### 3.1 订单处理流程

```
用户下单 → 订单状态:已付款(2) → 管理员确认订单 → 管理员点击发货 → 订单状态:已发货(3) → 用户确认收货 → 订单状态:已完成(4)
                                                              ↓
                                                   用户取消订单 → 订单状态:已取消(5)
```

### 3.2 商品管理流程

```
添加商品 → 填写商品信息 → 上传图片 → 选择分类 → 保存 → 商品上架
                                                    ↓
                                              编辑/删除/调整库存
```

## 4. 数据权限

| 角色 | 权限范围 |
| :--- | :--- |
| 超级管理员 | 所有功能权限 |
| 商品管理员 | 商品管理、分类管理 |
| 订单管理员 | 订单管理、订单统计 |
| 查看权限 | 仅查看，无修改删除权限 |

## 5. 界面原型说明

### 5.1 管理后台首页

- 顶部：系统名称、登录用户、退出按钮
- 左侧：功能菜单导航
- 主内容区：数据统计概览（今日订单、今日销售额、商品总数、用户总数）

### 5.2 列表页面通用布局

- 顶部：搜索框、筛选条件、新增按钮
- 中部：数据表格（支持分页）
- 底部：分页导航

### 5.3 表单页面通用布局

- 顶部：表单标题
- 中部：表单字段
- 底部：保存、取消按钮

## 6. 非功能需求

| 需求类型 | 描述 |
| :--- | :--- |
| 安全性 | 管理员密码加密存储，登录失败次数限制，操作日志记录 |
| 响应速度 | 页面加载时间不超过3秒 |
| 兼容性 | 支持主流浏览器（Chrome、Firefox、Safari） |
| 易用性 | 操作流程清晰，提供必要的提示信息 |

## 7. 与现有系统的集成

### 7.1 数据模型复用

| 现有实体 | 管理员功能使用场景 |
| :--- | :--- |
| User | 用户管理模块 |
| Goods | 商品管理模块 |
| Type | 分类管理模块 |
| Order | 订单管理模块 |
| OrderItem | 订单详情展示 |

### 7.2 状态码对应

| 订单状态码 | 状态名称 | 管理员操作 |
| :--- | :--- | :--- |
| 2 | 已付款 | 可发货 |
| 3 | 已发货 | 查看物流 |
| 4 | 已完成 | 无操作 |
| 5 | 已取消 | 无操作 |
